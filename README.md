# Password Strength Analyzer

A Python-based GUI application that evaluates password strength using cybersecurity best practices.

## Features

- Password strength analysis
- Show/Hide password option
- Password generator
- Entropy calculation
- Common password detection
- Password improvement suggestions
- Strength progress bar
- User-friendly Tkinter interface

## Technologies Used

- Python 3
- Tkinter
- Math Module
- String Module
- Random Module
